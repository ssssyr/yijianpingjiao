# 研究生评教自动化脚本

自动完成研究生课程评教的 Python 脚本，自动选择"完全同意"并提交。

## 功能说明

- 自动登录 UIS 账号
- 自动识别所有待评教课程
- 自动选择所有问题的"完全同意"选项
- 自动提交评教
- 支持暂存功能（已评教过的课程）
- 自动跳过已完成的课程

## 环境要求

- Python 3.x
- Chrome 浏览器
- ChromeDriver（与 Chrome 版本匹配）

## 安装步骤

### 1. 安装 Python

确保已安装 Python 3.x，可以通过以下命令检查：

```bash
python --version
```

如果没有安装，请访问 [Python官网](https://www.python.org/downloads/) 下载安装。

### 2. 安装 Selenium

```bash
pip install selenium
```

### 3. 下载 ChromeDriver

1. 查看你的 Chrome 版本：打开 Chrome → 地址栏输入 `chrome://version/`
2. 访问 [ChromeDriver下载页面](https://chromedriver.chromium.org/downloads)
3. 下载与你的 Chrome 版本匹配的 ChromeDriver
4. 将下载的文件解压，放到项目目录中

**注意**：本项目目录中已包含 `chrome-mac-arm64` 文件夹，如果是 Mac ARM 架构可以直接使用。

## 使用方法

### 基本使用

1. 打开终端/命令行，进入项目目录：

```bash
cd /your-project
```

2. 运行脚本：

```bash
python work.py
```

3. 按提示输入你的 UIS 学号和密码

4. 脚本会自动完成所有评教任务

### 修改用户名密码（可选）

如果不想每次输入用户名密码，可以修改 `work.py` 文件：

找到第 9-17 行，将注释取消：

```python

usrname = ''

# 如果需要交互式输入，可以取消下面的注释

```

## 执行流程

```
1. 打开评教网站
2. 输入 UIS 学号和密码
3. 自动登录
4. 关闭新手引导提示
5. 循环处理每门课程：
   - 点击课程卡片
   - 关闭引导提示
   - 点击所有"完全同意"选项
   - 点击"提交"按钮
   - 点击"确定"确认
   - 返回课程列表
6. 所有课程评教完成，自动退出
```

## 注意事项

1. **密码安全**：脚本中的密码是明文存储的，请注意保管代码文件，不要分享给他人
2. **网络连接**：确保网络连接稳定
3. **不要手动操作**：脚本运行期间不要手动操作浏览器
4. **评教内容**：默认选择"完全同意"，如需修改请编辑源码

## 常见问题

### Q1: 提示找不到 ChromeDriver

**A**: 确保 ChromeDriver 已下载并放在正确位置，或修改代码中的 ChromeDriver 路径。

### Q2: 提示 Chrome 版本不匹配

**A**: 更新 ChromeDriver 到与 Chrome 浏览器相同的版本。

### Q3: 脚本运行很慢

**A**: 正常现象，脚本需要等待页面加载。如果太慢可能是网络问题。

### Q4: 登录失败

**A**: 检查学号和密码是否正确，确认网络连接正常。

### Q5: 点击提交后没有反应

**A**: 可能是课程已经评教过了，脚本会自动跳过。
